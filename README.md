🤖 Python Chatbot

Developed as part of CodeAlpha Python Programming Internship

📌 Overview

A simple rule-based chatbot built using Python.
The chatbot interacts with users through the command line and responds to basic conversational inputs such as greetings, personal queries, and utility-based requests like date and time.

🚀 Features

🤖 Interactive command-line conversation
👤 Personalized responses using user name
💬 Handles basic inputs (hello, how are you, etc.)
⏰ Displays current time
📅 Displays current date
🆘 Help command support
👋 Exit functionality (bye)
⚠️ Handles unknown inputs gracefully

🛠️ Tech Used

Python
Datetime module

▶️ How to Run
git clone https://github.com/Rithika-B-7/CodeAlpha_Chatbot.git
cd CodeAlpha_Chatbot
python chatbot.py
